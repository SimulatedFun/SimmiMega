◆ getGlyphLookup()

GlyphLookup Glyphs::getGlyphLookup ( uint16_t  glyphId)

Gets a GlyphLookup object that tells you where in memory the glyph bitmap resides Also retrieves relevant information such as how wide/high the glyph is, etc. For glyph IDs, see asciiToGlyphLookupId(), but generally the structure is: 0 - 61 are lower/upper case letters and digits 62 - 82 are punctuation/ascii symbols 82+ are icons used for UI (e.g. pencil icon, eraser, trash bin)
